Boracle Marketplace

Profile Management - Arfan

In my role contributing to the development of the Boracle Web Application, I was tasked with several key responsibilities that significantly enhanced the application's functionality and user experience. My contributions include:

  • Database Querying: Executed complex queries to fetch product information from the Firebase database, demonstrating my capability in managing backend data interactions.
  • UI/UX Design: Undertook a complete overhaul of the product page's user interface, significantly improving its aesthetics and user engagement. This involved displaying product details such as Product Name, Category, and incorporating 'Learn More' buttons for enhanced user interaction.

Each of these components was developed with a focus on user experience, leveraging modern web development frameworks and techniques to create a seamless and engaging platform for Boracle's users.Additionally, I worked on the following features that were outside my major feature responsibilities:

  • Navigation Bars & Starting Page: Designed and implemented the application's navigation bars and starting page, ensuring a user-friendly and intuitive interface.
  • Health Needs Assessment: Developed a comprehensive health needs assessment tool that leverages React hooks (specifically, useState and useQuery). This feature calculates personalized health evaluations based on variables such as exercise frequency, sleep quality, and stress levels, providing users with valuable insights into their well-being.
  • User Profile & Settings: Created and refined the user profile and settings sections, allowing users to easily manage their personal information and preferences within the app.
  • Data Fields Organization: Worked on the structuring and organization of data fields across the application to ensure data integrity and ease of access.
