Boracle Marketplace

The Boracle Marketplace is a web application to promote the use and consolidate reliable information about smart wearable technology.

We used React.JS, Material UI, and Firestore to handle the bulk of the web application.

Boracle Marketplace Architecture

The Boracle Marketplace can be separated into three main components: the presentation, application, and database layers. Since we built our web application with React.JS, all three layers are handled through client side interaction. Information is queried from our web catalog on the user's end, and the resulting data is transformed into components to be displayed on the user's screen. This process is made secure by using Firestore security rules to limit the amount of access that users have to the database.
