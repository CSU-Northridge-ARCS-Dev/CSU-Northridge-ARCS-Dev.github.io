Reviews and Ratings - Matthew

The reviews and ratings system aims to bring transparency and a singular place to view information for smart wearable technology and related health applications. The following features were implemented for the review system:

  • Review summaries: Users are able to view the summary statistics for a product's reviews.
  • Submitting reviews: Users are able to submit a review for each product on the marketplace.
  • View external reviews: Users are able to click buttons to visit sites that have externally submitted review information (like Amazon).
  • Rating reviews: Users are able to use the thumbs up/thumbs down buttons to rate a view positively or negatively.
  • Reporting reviews: Users are able to report reviews that violate the terms of service.
  • Moderating reviews: System administrators can see all reports and remove review content that violates terms of service.

Additional work was done to create an automated system to webscrape sites for external reviews. This and the moderation feature need minimal work to be completed.
